Arrays

What are Arrays?

Arrays are linear data structures that store elements in contiguous memory locations.

Key Operations:

  • Insertion: O(n)
  • Deletion: O(n)
  • Access: O(1)
  • Search: O(n)

Linked Lists

What are Linked Lists?

Linked Lists are sequential data structures where elements are connected through references.

Key Operations:

  • Insertion: O(1)
  • Deletion: O(1)
  • Access: O(n)
  • Search: O(n)

Stacks

What are Stacks?

Stacks are linear data structures that follow the Last-In-First-Out (LIFO) principle. Think of it like a stack of plates - you can only add or remove plates from the top.

Key Operations:

  • Push (Insert): O(1)
  • Pop (Remove): O(1)
  • Peek (Top): O(1)
  • IsEmpty: O(1)

Queues

What are Queues?

Queues are linear data structures that follow the First-In-First-Out (FIFO) principle. Think of it like a line of people waiting - the first person to join the line is the first to leave.

Key Operations:

  • Enqueue (Insert): O(1)
  • Dequeue (Remove): O(1)
  • Front: O(1)
  • IsEmpty: O(1)

Trees

What are Trees?

Trees are hierarchical data structures with a root node and child nodes connected by edges. Binary trees, AVL trees, and Red-Black trees are common variants.

Key Operations:

  • Insertion: O(log n) for balanced trees
  • Deletion: O(log n) for balanced trees
  • Search: O(log n) for balanced trees
  • Traversal: O(n)

Graphs

What are Graphs?

Graphs are non-linear data structures consisting of vertices (nodes) and edges connecting these vertices. They can be directed or undirected, weighted or unweighted.

Key Operations:

  • Add Vertex: O(1)
  • Add Edge: O(1)
  • DFS Traversal: O(V + E)
  • BFS Traversal: O(V + E)

Heaps

What are Heaps?

Heaps are specialized tree-based data structures that satisfy the heap property. In a max heap, parent nodes are always greater than child nodes, while in a min heap, parent nodes are always smaller than child nodes.

Key Operations:

  • Insertion: O(log n)
  • Delete Max/Min: O(log n)
  • Get Max/Min: O(1)
  • Heapify: O(n)

Hash Tables

What are Hash Tables?

Hash tables are data structures that implement an associative array abstract data type, mapping keys to values using a hash function to compute an index into an array of buckets.

Key Operations:

  • Insertion: O(1) average
  • Deletion: O(1) average
  • Search: O(1) average
  • Worst case: O(n) with collisions
